### 5.3 技术架构

#### 5.3.1 架构图
```
┌───────────────────────────────────────────────────────────────────────────────────────────────┐
│                            自动化媒体处理管道 (AMP)                                           │
├───────────────────────┬───────────────────────┬───────────────────────────────────────────────┤
│  输入层               │  处理层               │  输出层                                    │
├───────────────────────┼───────────────────────┼───────────────────────────────────────────────┤
│ • 文件监控服务        │ • 预处理服务          │ • 存储服务                                 │
│ • 事件接收器          │ • AI增强服务          │ • 通知服务                                 │
│ • 批量任务调度        │ • 内容分析服务        │ • 报告生成器                               │
└───────────────────────┴───────────────────────┴───────────────────────────────────────────────┘
```

#### 5.3.2 服务边界与交互
- **输入**：
  - 文件系统事件（来自NAS监控）
  - 手动触发的处理请求
  - 批量处理任务
- **输出**：
  - 处理后的媒体文件
  - 处理报告
  - 分析结果和元数据
  - 通知事件
